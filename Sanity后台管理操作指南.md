## 一、通用说明

### 1.1 如何登录 Sanity 后台

1. 打开浏览器，访问 `https://tiancibaobei.sanity.studio`
2. 使用分配的 Sanity 账号登录
3. 登录后即可看到左侧的内容类型列表

### 1.2 后台界面概览

- **左侧边栏**：列出所有可编辑的内容类型（如"首页""新闻文章"等）
- **中央区域**：选中某个类型后，显示该类型下的所有文档列表
- **右侧编辑区**：点击某个文档后进入编辑界面，修改完成后点击右上角 **Publish**（发布）按钮

### 1.3 通用字段类型说明

| 字段类型 | 操作方式 | 说明 |
|---|---|---|
| **文本 (string)** | 直接输入文字 | 短文本，如标题、按钮文字 |
| **多行文本 (text)** | 在文本框内输入 | 长文本，支持换行 |
| **富文本 (Rich Text)** | 类似 Word 编辑器 | 支持加粗、标题、列表、插入图片和链接 |
| **图片 (image)** | 点击上传或拖入 | 上传后可裁剪热点区域 |
| **布尔 (boolean)** | 勾选/取消勾选 | 控制"是否显示""是否启用" |
| **数字 (number)** | 直接输入数字 | 如排序值、步骤编号 |
| **数组 (array)** | 点击"Add item"添加 | 可添加多条子项，支持拖动排序 |
| **引用 (reference)** | 从下拉列表中选择 | 关联其他文档，如新闻文章选择分类 |
| **Slug** | 自动生成或手动输入 | URL 路径标识，通常从标题自动生成 |
| **日期时间 (datetime)** | 通过日期选择器选择 | 文章发布时间等 |
| **链接 (url)** | 输入完整网址 | 如 `https://example.com` |

### 1.4 SEO 设置（通用组件）

几乎所有页面都有 **"SEO 设置"** 折叠区域，字段如下：

| 字段名称 | 说明 | 建议 |
|---|---|---|
| 页面标题（Meta Title） | 浏览器标签和搜索结果标题 | 50-60 个字符，包含核心关键词 |
| 页面描述（Meta Description） | 搜索结果摘要 | 120-160 个字符，简明描述页面内容 |
| 关键词 | 用标签形式添加 | 3-5 个核心关键词 |
| 社交分享标题（OG Title） | 分享到微信/微博时的标题 | 可留空，默认使用页面标题 |
| 社交分享描述（OG Description） | 分享到社交平台时的描述 | 可留空，默认使用页面描述 |
| 社交分享图片（OG Image） | 分享到社交平台时的图片 | **推荐尺寸：1200×630 px** |
| 规范链接（Canonical URL） | 防止重复内容 | 一般留空即可 |
| 禁止搜索引擎索引 | 勾选后页面不出现在搜索结果 | 一般不勾选 |

### 1.5 Banner 幻灯片（通用组件）

多个页面支持 **"Banner 轮播"**，每张幻灯片包含以下字段：

| 字段名称 | 说明 | 注意事项 |
|---|---|---|
| 桌面图片 | 电脑端显示的 Banner 图 | **必填**，推荐 2400×800 px，WebP/JPG 格式 |
| 移动端图片 | 手机端显示的 Banner 图 | 可选，推荐 900×600 px |
| Alt 文本 | 图片替代文字 | **必填**，简短描述图片内容 |
| 标题 | Banner 上的大标题 | 可选 |
| 副标题 | 标题下方的补充文字 | 可选 |
| 按钮文字 | Banner 上的按钮文案 | 如"立即咨询""了解更多" |
| 按钮链接 | 按钮点击后的跳转地址 | 如 `/start-your-journey` 或外部链接 |
| 启用 | 是否显示此张幻灯片 | 取消勾选则不显示 |

### 1.6 图片上传规范

#### 常见图片字段类型

本系统中有两种图片字段：

1. **图片（含替代文本）** — `imageWithAlt`
   - 包含：图片、替代文本（Alt）、图片说明（Caption）
   - **替代文本必填**，用于无障碍访问和 SEO
   - 图片说明可选，显示在图片下方

2. **普通图片** — `image`
   - 仅包含图片本身，无 Alt 字段
   - 用于二维码、Favicon 等

#### 通用图片规范

| 项目 | 要求 |
|---|---|
| **格式** | WebP（首选）、JPG、PNG（透明背景用 PNG） |
| **文件大小** | 内容图 500KB 以内，Banner 图 800KB 以内 |
| **色彩模式** | sRGB |
| **命名** | 使用有意义的英文名称，避免中文和特殊字符 |

### 1.7 咨询行动区 CTA（通用组件）

多个页面底部有 **"咨询入口/咨询行动区"**，字段如下：

| 字段名称 | 说明 | 示例 |
|---|---|---|
| 标题 | 行动区大标题 | "准备好了吗？" |
| 说明 | 标题下方的补充文字 | "联系我们，获取专业的助孕咨询服务。" |
| 按钮文字 | 按钮文案 | "开始咨询" |
| 按钮链接 | 按钮跳转地址 | `/start-your-journey` |

### 1.8 内容发布与生效

- 修改完成后，点击编辑器右上角的 **Publish** 按钮
- 发布后，网站会在 **1 分钟内** 自动更新（ISR 缓存刷新）
- 如果看不到更新，请清除浏览器缓存或等待几分钟
- **未发布的修改** 不会显示在网站上，可以先保存草稿再发布

## 十五、咨询记录

> **后台入口**：左侧边栏 → **咨询记录**  
> **说明**：用户通过网站咨询表单提交后自动创建，运营人员在此查看和跟进

### 查看咨询记录

每条咨询记录包含以下信息：

| 字段名称 | 说明 | 注意事项 |
|---|---|---|
| 姓名 | 客户姓名 | 必填 |
| 手机号码 | 客户手机 | 必填 |
| 微信号 | 客户微信 | 可选 |
| 所在城市 | 客户城市 | 可选 |
| 咨询需求 | 客户描述的需求 | 必填 |
| 方便联系时间 | 随时 / 上午 / 下午 / 晚上 | 必填 |
| 来源页面 | 提交来自哪个页面 | 自动记录，不可修改 |
| 处理状态 | 待跟进 / 跟进中 / 已完成 / 无效 | 需手动更新 |
| 提交时间 | 提交时间 | 自动记录，不可修改 |

### 跟进操作

1. 在列表中找到待跟进的记录
2. 点击进入详情页
3. 查看客户信息和需求
4. 将 **处理状态** 改为"跟进中"
5. 联系客户后，将状态改为"已完成"或"无效"
