# 天赐宝贝网站后台使用说明

本文档说明 Sanity Studio 后台每个字段在网站前台的对应位置。

---

## 一、站点设置

> 路径：**站点设置**

此为全局配置，影响网站所有页面。

### 1.1 品牌信息

| 后台字段 | 字段名 | 网站位置 | 说明 |
|---------|--------|---------|------|
| 网站名称 | `siteName` | Header 左上角 Logo 文字（无 Logo 图时）、页脚品牌名、浏览器标签默认标题 | 必填 |
| 网站描述 | `description` | 首页 SEO 默认描述、OG 分享描述回退值 | — |
| 网站 Logo | `logo` | Header 左上角图片 Logo | 建议 400×80px PNG 透明底 |
| 默认分享图 | `defaultShareImage` | 微信/社交平台分享时的默认封面 | 建议 1200×630px |

### 1.2 联系方式

| 后台字段 | 字段名 | 网站位置 | 说明 |
|---------|--------|---------|------|
| 联系电话 | `phone` | 所有页面底部 CTA 按钮、页脚、"踏上为人父母之旅"页面电话卡片、移动端拨号链接 | — |
| 服务时间 | `serviceHours` | 页脚、"踏上为人父母之旅"页面电话卡片下方 | 如"周一至周日 9:00-18:00" |
| 微信咨询二维码 | `wechatQrCode` | 移动端底部"微信咨询"弹窗中显示 | 建议 500×500px |
| 微信公众号二维码 | `wechatPublicQrCode` | 桌面端右侧悬浮"微信公众号"按钮的浮层中显示 | 建议 500×500px |

### 1.3 咨询栏文字

| 后台字段 | 字段名 | 网站位置 | 说明 |
|---------|--------|---------|------|
| 侧边栏咨询文字 | `sidebarCtaText` | 桌面端右侧悬浮按钮（预留） | — |
| 移动咨询栏文字 | `mobileCtaText` | 移动端底部固定栏（预留） | — |

### 1.4 页脚信息

| 后台字段 | 字段名 | 网站位置 | 说明 |
|---------|--------|---------|------|
| 页脚简介 | `footerDescription` | 页脚左侧品牌描述文字 | 默认："专注助孕咨询服务…" |
| 备案号 | `icpNumber` | 页脚右下角 | 如"京ICP备XXXXXXXX号" |
| 版权文字 | `copyrightText` | 页脚底部左侧 | 默认：© 年份 + 品牌名 |

### 1.5 默认 SEO

> 影响所有页面的 SEO 回退值

| 后台字段 | 字段名 | 说明 |
|---------|--------|------|
| 页面标题 | `seo.metaTitle` | 各页面 SEO 缺失时的回退标题 |
| 页面描述 | `seo.metaDescription` | 各页面 SEO 缺失时的回退描述 |
| 关键词 | `seo.keywords` | 全站默认关键词，各页面可覆盖 |
| OG 标题 | `seo.ogTitle` | 社交分享标题回退值 |
| OG 描述 | `seo.ogDescription` | 社交分享描述回退值 |
| OG 图片 | `seo.ogImage` | 社交分享图片回退值 |
| 规范链接 | `seo.canonicalUrl` | — |
| 禁止索引 | `seo.noIndex` | 勾选后全站默认 noindex |

### 1.6 默认 Banner

> 当某个页面没有设置自己的 Banner 时，使用此默认 Banner

| 后台字段 | 字段名 | 网站位置 | 说明 |
|---------|--------|---------|------|
| 桌面图片 | `defaultBanner.desktopImage` | 无页面 Banner 时的全站默认 Banner | 1920×600px |
| 移动图片 | `defaultBanner.mobileImage` | 同上，移动端 | 750×500px |
| Alt 文本 | `defaultBanner.alt` | 图片替代文本 | 必填 |
| 标题 | `defaultBanner.title` | Banner 上覆盖的大标题 | — |
| 副标题 | `defaultBanner.subtitle` | 标题下方说明文字 | — |
| 按钮文字 | `defaultBanner.buttonText` | Banner 上的按钮 | — |
| 按钮链接 | `defaultBanner.buttonLink` | 按钮跳转地址 | — |
| 启用 | `defaultBanner.isActive` | 是否显示 | — |

### 1.7 导航显示文字

> 可修改导航栏显示文字，但**不能修改路由、顺序或增加一级页面**

| 后台字段 | 默认值 |
|---------|--------|
| 首页 | 首页 |
| 关于准父母 | 关于准父母 |
| 助孕流程 | 助孕流程 |
| 新闻资讯 | 新闻资讯 |
| 为什么选择我们 | 为什么选择我们？ |
| 常见问题 | 常见问题 |
| 踏上为人父母之旅 | 踏上为人父母之旅 |

## 二、固定页面

> 路径：**固定页面 → [页面名]**

每个固定页面都有独立的 Banner、模块内容和 SEO 设置。

---

### 2.1 首页

> 路径：**固定页面 → 首页**

#### Banner 轮播

同站点设置的 Banner 字段说明。可上传多张，自动轮播。

#### 品牌简介

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 品牌简介标题 | `brandIntroTitle` | 首页第一个白色区块的大标题 |
| 品牌简介内容 | `brandIntroContent` | 标题下方的描述文字 |
| 品牌简介图片 | `brandIntroImage` | （预留） |

#### 第三代试管服务介绍

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 服务标题 | `ivfTitle` | 浅青色区块标题，如"第三代试管婴儿技术" |
| 服务描述 | `ivfDescription` | 标题下方描述 |
| 服务特点 | `ivfFeatures[]` | 标题下方的卡片网格（1-3列） |
| ↳ 标题 | `feature.title` | 卡片标题 |
| ↳ 描述 | `feature.description` | 卡片描述 |
| ↳ 图标 | `feature.icon` | （预留，当前统一使用婴儿图标） |

> 标题和描述均不填时，此整个区块不显示。

#### 核心优势

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 标题 | `advantagesTitle` | 浅青色区块标题，默认"为什么选择我们" |
| 优势列表 | `advantages[]` | 4 列卡片网格 |
| ↳ 标题 | `advantage.title` | 卡片标题 |
| ↳ 描述 | `advantage.description` | 卡片描述 |
| ↳ 图标 | `advantage.icon` | 可选：users / heart / shield / award |
| ↳ 颜色 | `advantage.color` | 可选：pink / purple / yellow / green |

#### 助孕流程摘要

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 标题 | `journeyTitle` | 白色区块标题，默认"助孕流程" |
| 流程步骤 | `journeySteps[]` | 5 列步骤圆点 |
| ↳ 标题 | `step.title` | 步骤名称 |
| ↳ 描述 | `step.description` | 步骤说明 |
| ↳ 步骤编号 | `step.stepNumber` | 圆点中的数字 |

#### 服务数据

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 标题 | `statsTitle` | 蓝色背景区块标题，默认"我们的成绩" |
| 数据列表 | `stats[]` | 4 列数字展示 |
| ↳ 数值 | `stat.value` | 大号数字，如"10+" |
| ↳ 标签 | `stat.label` | 数字下方文字，如"年行业经验" |

#### 新闻推荐

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 标题 | `newsTitle` | 浅青色区块标题，默认"新闻资讯" |
| 推荐数量 | `featuredNewsCount` | 控制显示几篇（默认 3） |

> 实际显示的是 Sanity 中勾选了"推荐"的新闻文章，按发布时间倒序。

#### 常见问题精选

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 标题 | `faqTitle` | 白色区块标题，默认"常见问题" |
| 精选数量 | `featuredFaqCount` | 控制显示几条（默认 5） |

> 实际显示的是 Sanity 中勾选了"推荐"的 FAQ，按排序升序。

#### 咨询行动区

| 后台字段 | 字段名 | 网站位置 |
|---------|--------|---------|
| 标题 | `cta.title` | 蓝色背景区块标题 |
| 说明 | `cta.description` | 标题下方描述 |
| 按钮文字 | `cta.buttonText` | 右侧按钮文字 |
| 按钮链接 | `cta.buttonLink` | 按钮跳转地址 |

#### SEO 设置

同站点设置的 SEO 字段说明。页面 SEO 优先于站点默认 SEO。

| 后台字段 | 字段名 | 网站位置 | 说明 |
|---------|--------|---------|------|
| 标题 | `title` | 新闻列表卡片标题、详情页 H1 | 必填 |

## 五、咨询记录

> 路径：**咨询记录 → 全部记录 / 待跟进 / 跟进中**

此为后台管理区域，**不出现在前台网站**。记录由用户提交表单后自动创建。

| 后台字段 | 说明 |
|---------|------|
| 姓名 | 用户填写的姓名 |
| 手机号码 | 用户填写的手机号 |
| 微信号 | 选填 |
| 所在城市 | 选填 |
| 咨询需求 | 用户描述的需求 |

## 六、Banner 图片规范

## 八、SEO 回退逻辑

每个页面的 SEO 字段按以下顺序回退：

```
页面 SEO 字段 → 页面内容标题/描述 → 站点设置默认 SEO → 代码硬编码默认值
```

示例：如果"关于准父母"页面未设置 `seo.metaDescription`，则使用 `siteSettings.defaultSeo.metaDescription`，再未设置则使用代码中的默认描述。
